Extend Database Expiry Date

Extend your Odoo database expiration date easily.

Overview

This module allows Odoo Enterprise users to automatically extend the expiration date of their database each month. This can help in managing demo or development databases without manual intervention.

Features

  • Automatic extension of database expiry date by one month.
  • Configurable scheduler to run at specific intervals.
  • Easy to install and requires no additional configuration.

Installation

Simply install the module through the Odoo Apps interface. Once installed, the scheduler will automatically adjust the database's expiry date.

Requirements

This module requires Odoo Enterprise Edition as it modifies the enterprise-only database expiry parameter.
